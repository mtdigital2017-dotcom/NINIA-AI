# Planner v1

## Propósito
Convertir una decisión operativa en un plan con tareas, dependencias, responsables y criterios de aceptación.

## Entradas
Resultado estructurado del Decision Engine.

## Salidas
Archivo JSON trazable en `NINIA_OS/PLANS/`.

## Límites
No ejecuta tareas, no modifica decisiones y no aprueba evidencia.

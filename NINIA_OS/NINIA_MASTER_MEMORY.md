# 0. Propósito

Este documento es la **memoria oficial del proyecto NINIA**.

A partir de esta versión:

-   Ninguna decisión estratégica dependerá únicamente de conversaciones.
-   Toda decisión arquitectónica deberá consolidarse aquí.
-   Este documento es la fuente oficial de verdad para el desarrollo.

------------------------------------------------------------------------

# 1. Identidad

**NINIA** es una Plataforma Mundial de Inteligencia Científica para la
Protección Integral de Niñas, Niños y Adolescentes en Entornos
Digitales.

No es un chatbot. No es un observatorio tradicional. No es un
repositorio documental.

Transforma evidencia verificable en inteligencia, decisiones, políticas
públicas, cooperación y contenidos especializados.

------------------------------------------------------------------------

# 2. ADN (Principios permanentes)

-   Plataforma mundial.
-   Multilingüe desde el diseño.
-   El documento original nunca se modifica.
-   Toda traducción conserva vínculo con el original.
-   Toda afirmación debe tener evidencia.
-   Toda evidencia debe ser trazable.
-   Todo contenido deriva de un único objeto de conocimiento.
-   No se crean componentes duplicados.
-   Antes de desarrollar se audita el estado del proyecto.
-   Validación humana para incorporar conocimiento oficial.

# 6. Multilenguaje

Toda la plataforma será multilingüe.

Siempre se conserva:

-   idioma original;
-   documento original;
-   citas originales;
-   bibliografía original.

Las traducciones son capas derivadas y nunca sustituyen la fuente.

# Registro de aprendizaje

## 2026-07-11

### Conceptos

- **Commit:** registro local de un conjunto de cambios.
- **Push:** envío de commits desde el computador hacia GitHub.
- **GitHub Actions:** servicio que ejecuta tareas automáticas después de eventos como un push.
- **Secret:** valor privado cifrado disponible para el workflow, pero no guardado en el código.
- **Hash:** huella digital que permite verificar que un archivo no cambió.
- **Workflow:** secuencia automática de pasos.
- **Cuenta de servicio:** identidad técnica usada por una automatización.
- **Unidad compartida:** espacio de Google Workspace propiedad de la organización, no de una persona.

### Regla práctica

No hacer `git push` hasta revisar `git status` y ejecutar las pruebas.
